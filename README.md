# javascript-web-jquery-canvas-capatcha-image-wave-noise

## Description
A demo of image based Capatcha
using canvas with wave noise.

## Tech stack
- jquery
- bootstrap
- canvas

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
